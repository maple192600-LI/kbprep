# Public Templates

This directory contains only generic or sanitized templates that are safe to publish.

Private course, platform, source-brand, or personal cleanup templates belong in:

```text
.kbprep/rules/templates/
```

For example, a local `curated_obsidian_kb` setup may provide:

```text
.kbprep/rules/templates/obsidian_course_kb.json
```

The local private template overrides packaged public templates with the same name.
