Target Gaps

  • Canonical IR is not yet the universal worker artifact.
  • Route-native source spans, relationships, assets, annotations, and IR-based regeneration remain incomplete.
  • Selective rerun and playlist workflows remain partial: failed/pending batch rerun, explicit playlist input, and playlist rerun evidence preservation exist, while policy/CIR affected targeting remains open.
  • Optional media and YouTube routes are partial and need broader real-sample, dependency, timeout, and transcript-quality evidence before verified promotion.

Verification

  • npm run check:flowchart verifies HTML and JSON flowchart alignment.
  • npm run check:development-docs verifies the current stage document set.
  • npm run pack:check verifies governance, policy, package, and capability guardrails.
